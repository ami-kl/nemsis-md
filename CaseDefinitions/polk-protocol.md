# Medical Quality Control: Vital Signs and Procedures

## Vital Signs Abnormal Parameters

| Parameter | Abnormal Range |
|-----------|----------------|
| **Temperature** | < 96°F or > 100°F |
| **Heart Rate** | < 50 bpm or > 120 bpm |
| **Respiratory Rate** | < 10 breaths/min or > 30 breaths/min |
| **EtCO2** | < 30 mmHg or > 50 mmHg |
| **SpO2** | < 92% |
| **Systolic Blood Pressure (SBP)** | < 90 mmHg or > 200 mmHg |
| **Mean Arterial Pressure (MAP)** | < 70 mmHg |
| **Glasgow Coma Scale (GCS)** | < 15 or new onset neurologic deficit |
| **AVPU** | Any abnormal response |
| **Blood Glucose** | < 50 mg/dL or > 300 mg/dL |
| **Blood Pressure (Pregnant Patients)** | SBP > 160 mmHg or DBP > 110 mmHg |
| **Rankin Score** | > 3 |

## Procedures Performed

- **Cardiovascular**:
  - EKG transmission
  - Pacing
  - Defibrillation
  - Cardioversion
  - Chest Compressions / Lucas device
  - Blood products administration

- **Airway Management**:
  - Oropharyngeal Airway (OPA) / Nasopharyngeal Airway (NPA)
  - Intubation / Supraglottic Airway (SGA)
  - Cricothyroidotomy
  - CPAP / Non-Rebreather Mask (NRB) / Bagging
  - McGill Forceps
  - Ventilation placement

- **Trauma/Orthopedic**:
  - Needle decompression
  - Chest Thoracostomy
  - Chest Seal
  - Traction Splint
  - Tourniquet / Kendrick Extrication Device (KED)

- **Other**:
  - Orogastric (OG) / Nasogastric (NG) tube placement
  - Intraosseous (IO) placement
